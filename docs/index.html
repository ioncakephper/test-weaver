Effortless Test Case Generation.

TestWeaver is a command-line tool that generates comprehensive test cases from simple YAML configurations.

Get Started View on GitHub

Features

Fast Test Generation

Quickly generate boilerplate test files and code from a simple configuration.

Language Agnostic

Works with any programming language, including JavaScript and TypeScript, by using custom templates for your test files.

Flexible Configuration

Define test suites, units, and custom variables with a simple YAML file.

How to Use TestWeaver

1. Install the CLI

Install TestWeaver globally via npm. 

npm install -g test-weaver

2. Create Your YAML

Define your test suites and properties in a `your-file.yml`. 

When CLI Command:
     Gets an empty line:
            A default command defined:
                 - execute the command
                 - return no errors code
            No default command defined:
                 - display help screen
                 - returns no error code
     Gets flag -h or --help:
          - display help screen
          - return no errors code

3. Run the Generator

Execute the `generate` command with your YAML file. 

testweaver generate -f your-file.yml

4. See the Generated Test File

TestWeaver creates a `your-file.test.js` file with a structure that mirrors your YAML configuration. 

describe('When CLI Command', () => {
      describe('Gets an empty line', () => {
               describe('A default command defined', () => {
                    it.todo('execute the command');
                    it.todo('return no errors code');
               })
      })
      describe('Gets flag -h or --help', () => {
                it.todo('display help screen');
                it.todo('return no errors code');
      })
})
